🎮 Консольная игра «Города»

📌 О проекте

m4-2-python- — это консольная реализация классической игры «Города», написанная на Python.

Игрок и компьютер по очереди называют города.
Каждый следующий город должен начинаться на последнюю букву предыдущего (с учётом исключений).

Проект реализован без сторонних библиотек — используется только стандартная библиотека Python.

⸻

🧠 Логика игры
 • Игра начинается со случайного города, выбранного компьютером.
 • Игрок вводит название города в консоль.
 • Город должен:
 • существовать в списке cities.txt
 • не быть использован ранее
 • начинаться на правильную букву
 • После хода игрока компьютер делает свой ход.
 • Если компьютер не может найти подходящий город — игрок побеждает.
 • У игрока есть 5 попыток на ошибочные вводы.
 • После исчерпания попыток игра завершается поражением.

❗ Исключаемые буквы

Если город заканчивается на: ь, ъ, ы; 
то используется предпоследняя буква.

⚙️ Технологии
 • Python 3.13
 • Стандартная библиотека (random)
 • Работа с файлами (UTF-8)

🚀 Запуск проекта
 1. Убедитесь, что установлен Python 3.13
 2. Склонируйте репозиторий: git clone https://github.com/ВАШ_НИКНЕЙМ/m4-2-python-.git

3. Перейдите в папку проекта: cd m4-2-python-

4. Запустите игру:
python main.py

🎯 Пример запуска:
Ход компьютера: Москва
Ваш ход: астрахань
Ход компьютера: Нижний новгород
Ваш ход: ...

🛠 Реализованный функционал
 • Загрузка городов из файла
 • Проверка корректности ввода
 • Исключение повторов
 • Автоматический выбор города компьютером
 • Ограничение количества попыток
 • Обработка специальных букв
 • Хранение использованных городов через set()
